# Rules

These are hard filters. If a job matches any of these rules, filter it out entirely.

1. Skip any listing that requires 5 or more years of professional frontend or software engineering experience. Roles at the "junior," "mid-level," or "1-3 years" level pass. Roles that say "3+ years" pass. Roles that clearly expect senior-level scope, staff-level ownership, or team leadership should be skipped.
2. Skip any role that is fully backend with no frontend component. If the listing mentions zero UI, client-side, or frontend work, filter it out.
3. Skip roles that are primarily DevOps, SRE, data engineering, or machine learning — even if they mention JavaScript or TypeScript in passing.
